<h1>Todo List RESTful API</h1>

<h3>使用工具</h3>
<ul>
    <li>Express</li>
    <li>CORS</li>
    <li>JWT</li>
    <li>UUID</li>
    <li>MongoDB</li>
</ul>

<h3>功能</h3>
<ul>
    <li>開發 RESTful API 並由 Vue 前端串接</li>
    <li>註冊帳號與登入帳號，並串接資料庫</li>
    <li>儲存每個會員的待辦事項</li>
    <li>使用 MongoDB Atlas 實現 CRUD 功能</li>
    <li>登入系統採用 JWT 驗證</li>
</ul>
